# Leica Camera Enabler (Safe)

A lightweight Magisk/KernelSU module to enable Leica features for Xiaomi 17 Ultra's Camera app

## Features

- Sets `ro.theme_customize=LCC`
- Sets `ro.theme_leica=true`
- Sets `camera.debug.safe.check.disable=true`

## Installation

1. Download the zip file from the [Releases](https://github.com/mexusbg/Nezha-Leica-Camera-Enabler/releases) section.
2. Install via Magisk or KernelSU manager.
3. Reboot.
4. Clear Camera app data
